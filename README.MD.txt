## This is a program that takes the user input to determine which country would be 
the best fit for their next vacation. Once the user submits the form, the program
will instantly display the country that is the best fit for their next vacation.

## Setup/Installation Requirements

* _Clone repository_
* _Open repository index in web browser _
* _Open webpage_
* _For each of the five questions, select an answer that best fits you. 
Press the submit button to display the country that is your best fit. _

Specifications:

Website uses a point system to determine which country is the best fit for the user. Each answer gives a point to at least one country.
The country with the most points will be displayed.
Example input: Summer, Splurge, Nature, Modern, Hate them.
Example output: Iceland
If two or more countries share the same  ending point value all countries will be displayed. 
Example input: Summer, Average, Nature, Historic, Like them
Example output: Iceland, Mexico, India
